# Event-driven Architecture (event-driven-architecture)
Event-driven architecture (EDA) is a software architecture pattern promoting the production, detection, consumption of, and reaction to events. In this pattern, systems are designed to respond to state changes asynchronously through event messages. EDA enables loose coupling between services and is widely used in microservices, real-time data processing, and distributed systems.
